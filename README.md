# Zozoworld

Zozoworld is a campus marketplace built for Al-Hikmah University students.

## Features
- Browse vendors by category
- Find food, fashion, and services
- Contact vendors directly via WhatsApp

## Categories
- Food
- Fashion
- Accessories
- Cakes & Treats
- Perfume & Skincare
- Beauty Services

## Goal
To make it easy for students to discover and connect with trusted vendors on campus.
